# [초보자용] Windows에서 NotebookLM-py 시작하기 가이드

이 가이드는 컴퓨터 터미널(검은 창) 사용이 낯선 분들을 위해, Windows 환경에서 Python을 설치하고 NotebookLM 기능을 사용하는 방법을 아주 쉽게 설명합니다.

---

## 1단계: Python(파이썬) 설치하기

컴퓨터가 NotebookLM 도구를 이해할 수 있도록 '파이썬'이라는 프로그램을 먼저 설치해야 합니다.

1.  **공식 사이트 접속**: [python.org/downloads](https://www.python.org/downloads/)에 접속합니다.
2.  **다운로드**: 노란색 **[Download Python 3.xx.x]** 버튼을 클릭하여 설치 파일을 받습니다.
3.  **설치 시 주의사항 (가장 중요!)**:
    *   다운로드된 파일을 실행합니다.
    *   설치 창 하단에 있는 **[Add Python 3.xx to PATH]** 체크박스를 **반드시 체크**하세요. (이걸 안 하면 명령어를 인식하지 못합니다.)
    *   그다음 **[Install Now]**를 클릭하여 설치를 완료합니다.

---

## 2단계: 터미널(명령 프롬프트) 열기

명령어를 입력하기 위해 윈도우의 '명령 프롬프트'를 실행합니다.

1.  키보드의 **[Windows 로고 키]**를 누릅니다.
2.  `cmd` 라고 입력합니다.
3.  **[명령 프롬프트]** 앱을 클릭하여 실행합니다. (검은색 창이 나타납니다.)

---

## 3단계: NotebookLM 도구 설치하기

검은색 창(터미널)에 아래 명령어들을 한 줄씩 복사해서 붙여넣고 **[Enter]** 키를 누르세요.

1.  **도구 설치**:
    ```cmd
    pip install "notebooklm-py[browser]"
    ```
    *(글자들이 빠르게 지나가며 설치가 진행됩니다. 잠시 기다려 주세요.)*

2.  **브라우저 설치**:
    ```cmd
    playwright install chromium
    ```
    *(인증을 위해 필요한 과정입니다.)*

---

## 4단계: 내 계정 연결하기 (로그인)

이제 본인의 구글 계정을 이 도구와 연결해야 합니다.

1.  검은 창에 아래 명령어를 입력합니다:
    ```cmd
    notebooklm login
    ```
2.  잠시 후 자동으로 크롬 브라우저 창이 열립니다.
3.  평소처럼 구글 계정으로 로그인을 완료합니다.
4.  로그인이 완료되었다는 메시지가 뜨면 브라우저 창을 닫아도 됩니다.

---

## 5단계: 간단 사용법

이제 모든 준비가 끝났습니다! 아래 명령어들로 NotebookLM을 활용해 보세요.

### 내 노트북 목록 보기
내가 만든 노트북들이 무엇이 있는지 확인합니다.
```cmd
notebooklm list
```

### 특정 노트북에 질문하기
(예: '나의 일기'라는 노트북이 있다면)
```cmd
notebooklm ask --notebook "나의 일기" "이 내용의 핵심을 세 줄로 요약해줘"
```

### 오디오(팟캐스트) 생성하기
내가 넣은 자료들을 바탕으로 대화형 오디오를 만듭니다.
```cmd
notebooklm generate audio --wait
```

---

## 💡 팁
- **복사/붙여넣기**: 터미널에서 마우스 오른쪽 버튼을 클릭하면 복사한 내용이 붙여넣기 됩니다.
- **명령어 오류**: 만약 "명령어를 찾을 수 없습니다"라는 오류가 뜨면, 컴퓨터를 한 번 껐다 켜보세요. (설치된 설정이 적용되기 위함입니다.)

이제 여러분도 터미널을 통해 강력한 NotebookLM 기능을 사용하실 수 있습니다!
