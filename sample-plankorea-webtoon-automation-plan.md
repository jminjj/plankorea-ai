# [Harness 기반] 플랜코리아 뉴스 -> 웹툰 자동화 마스터 플랜

이 계획은 **Harness(하네스) 프레임워크**를 통해 설계된 전용 AI 에이전트들을 활용하여, 초보자도 전문가 수준의 인스타그램 웹툰을 제작할 수 있도록 돕는 최첨단 자동화 가이드입니다.

---

## 1. 에이전트 팀 구성 (Harness Team)

자동화를 위해 당신을 도울 두 명의 전문 에이전트를 소개합니다.

| 에이전트 | 역할 | 초보자를 위한 설명 |
| :--- | :--- | :--- |
| **초보자 도우미**<br>(Beginner Assistant) | 가이드 및 인터페이스 | "어려운 용어를 쉬운 말로 풀어서 설명해주고, 전체 과정을 친절하게 안내합니다." |
| **웹툰 아키텍트**<br>(Webtoon Architect) | 콘텐츠 분석 및 설계 | "뉴스를 읽고 재미있는 웹툰 대본을 쓰고, AI가 그림을 잘 그릴 수 있게 지시문을 만듭니다." |

---

## 2. 하네스 기반 자동화 워크플로우

### 단계 1: 프로젝트 시작 (초보자 도우미 호출)
터미널이나 AI 채팅창에서 다음과 같이 말하세요.
> "안녕, 플랜코리아 뉴스로 웹툰을 만들고 싶어. 시작을 도와줘."
*   **AI의 반응**: 초보자 도우미가 나타나 필요한 준비물(뉴스 링크 등)을 확인하고, 전체 단계를 친절히 안내합니다.

### 단계 2: 뉴스 분석 및 대본 설계 (웹툰 아키텍트 가동)
링크를 전달하면 웹툰 아키텍트가 작업을 수행합니다.
*   **작업 내용**: 
    1. 뉴스 본문에서 '감동 포인트' 추출.
    2. 인스타그램용 8컷 스토리보드 작성.
    3. 일관된 캐릭터 묘사 및 화풍(Style) 프롬프트 생성.
*   **결과물**: 각 컷별 이미지 생성 지시문(프롬프트) 리스트.

### 단계 3: 이미지 생성 및 검토
웹툰 아키텍트가 제공한 지시문을 복사하여 이미지 생성 AI(DALL-E 3 등)에 넣습니다.
> "웹툰 아키텍트가 준 1번 컷 지시문을 실행해줘."

### 단계 4: 최종 편집 및 브랜드 입히기 (Canva 활용)
이미지가 완성되면 캔바(Canva)를 통해 마무리합니다.
*   **초보자 팁**: 캔바의 '데이터 대량 제작' 기능을 사용하여 에이전트가 만든 대본을 한 번에 모든 이미지에 입히는 방법을 초보자 도우미가 설명해 드립니다.

---

## 3. 핵심 자동화 도구 세트

1.  **Harness Agents**: `.claude/agents/`에 정의된 에이전트들이 복잡한 설계를 대신합니다.
2.  **Gemini CLI / ChatGPT**: 에이전트들이 활동하는 작업 공간입니다.
3.  **DALL-E 3 / Midjourney**: 고품질 이미지를 생성하는 화가 역할을 합니다.
4.  **Canva**: 마지막 포장을 담당하는 디자인 도구입니다.

---

## 4. 하네스 방식의 장점

- **일관성**: `webtoon-architect`가 설정한 프롬프트 덕분에 1컷부터 8컷까지 주인공의 얼굴과 그림체가 일정하게 유지됩니다.
- **전문성**: NGO의 가치(자립, 연대)를 이해하는 에이전트가 시나리오를 써서 내용의 깊이가 다릅니다.
- **편의성**: 사용자는 "설계해줘", "그려줘" 같은 자연어만 사용하면 됩니다.

---

## 5. 바로 시작하기

1.  `.claude/agents/` 폴더를 확인하여 에이전트들이 준비되었는지 보세요.
2.  **초보자 도우미**에게 인사를 건네며 플랜코리아 뉴스 링크를 전달하세요.
3.  에이전트들이 제안하는 단계별 지시를 따라오기만 하면 됩니다!
