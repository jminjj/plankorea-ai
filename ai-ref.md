# AI 이용 help resource (ai_ref.md)

## gemini cli 설치

### nvm 설치 (nvm, node version 관리 프로그램) 
- 프로그램 다운로드 : [https://github.com/coreybutler/nvm-windows/releases](https://github.com/coreybutler/nvm-windows/releases)
  - 버전별 섹션 하단 : nvm-setup.exe 다운로드 받아서 설
- nvm 참고 : [https://github.com/coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows)

### node 설치 
- powershell 관리자 권한으로 실행
- node 최선버전 설치
```
nvm install latest
nvm list
nvm use <version>
```
- nvm 참고 Usage 섹션 참고 : [https://github.com/coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows?tab=readme-ov-file#usage)

### gemini cli 설치
- 설치: [https://geminicli.com/docs/get-started/installation/](https://geminicli.com/docs/get-started/installation/#install-gemini-cli)
```
npm install -g @google/gemini-cli
```
